Welcome to the Object Editor!
Select the sprite associated with the object from the
top menu bar.
Add events to the object by either double clicking on 
the event in the left pane or selecting the event and 
clicking the add button.
To edit a previously added event, either double click 
on the event or select the event and click the edit 
button.
To delete a previously added event, select the event
and click the delete button.
Rename the object using the text box located in the 
bottom right of the window.
Choose whether your object is solid (can be passed 
through or not) by clicking the checkbox in the bottom 
pane.
